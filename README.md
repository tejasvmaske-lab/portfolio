# 🌌 Cinematic Portfolio Website

A modern cinematic developer portfolio built using pure HTML, CSS, JavaScript, and GSAP — focused on smooth interactions, premium motion design, and a minimal futuristic aesthetic.

This portfolio is designed to feel less like a traditional resume website and more like an interactive digital identity.

---

# ✨ Features

- 🎭 Interactive avatar with mouse-follow movement
- 🌟 Dynamic glowing background effect
- 🎬 Cinematic GSAP entrance animations
- 💬 Hover-based microinteractions
- 🖱️ Custom animated cursor
- 🌑 Minimal dark premium UI
- ⚡ Smooth and lightweight experience
- 📱 Responsive structure (work in progress)

---

# 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)

---

# 🎯 Design Philosophy

The goal of this portfolio is to create a clean, immersive, and memorable first impression while maintaining simplicity and performance.

Instead of overwhelming the user with excessive animations or effects, the website focuses on:
- subtle motion
- cinematic transitions
- minimal interactions
- premium spacing and composition
